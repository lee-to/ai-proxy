# Support

Use the following channels:

- Bug reports: GitHub Issues at <https://github.com/lee-to/ai-proxy/issues>
- Feature requests: GitHub Issues or Discussions, if enabled
- Security reports: <thecutcode@gmail.com>, following [SECURITY.md](SECURITY.md)

When asking for help, include:

- OS and architecture.
- `ai-proxy` version or commit.
- Relevant config with secrets removed.
- Exact command and error output.
- Whether you are using base-url mode, blind CONNECT tunneling, or MITM mode.

Do not post real API keys, subscription tokens, private prompts, or captured traffic.
